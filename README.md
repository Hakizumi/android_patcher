## 🚀 Quick Start
### Download patched software
Recommended for newbies or people who have never been exposed to the **LSPosed**.

Download latest `xxx_someversion_patched.apk` and install it on your **Android** mobile phone.
Then you can use the patched application normally as official software with the same features.

### Patch the application yourself with `LSPosed` framework
Download latest patch pack from [release page](https://github.com/Hakizumi/android_patcher/releases),
then the **LSPosed** framework will scan the patch pack as a **LSPosed** module,
then patch the app corresponding to the patch pack in **LSPosed**.
